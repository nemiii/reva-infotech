# Reva Infotech Website Template Document

## Purpose

This document defines the positioning, content direction, and template intent for [index2.html](/Users/nemiamin/Documents/forgea-web/index2.html:1), which serves as a premium marketing website template for **Reva Infotech**.

Reva Infotech is positioned as a modern software company that:

- builds personal applications powered by AI and intelligent automation
- designs and develops custom software for startups, growth-stage teams, and enterprises
- delivers tailored digital products for businesses that need modern, scalable, user-friendly software

The current `index2.html` template uses a refined, editorial SaaS visual language inspired by the screenshot references in `landing.md`, while this document aligns the business messaging more closely to the service model seen on AppIt Ventures.

## Brand Positioning

### Core positioning

Reva Infotech builds personal software experiences and custom digital products for the modern world.

### Expanded positioning

Reva Infotech helps founders, teams, and businesses design, build, customize, and scale powerful software applications across web, mobile, AI-assisted, and internal business systems.

### Brand promise

We create software that feels intuitive for people, performs reliably for businesses, and is tailored to the way modern teams actually work.

## What Reva Infotech Does

Reva Infotech operates across two connected business lines:

### 1. Product development

Reva develops personal applications and intelligent software products that help individuals and teams:

- capture and organize information
- automate repetitive work
- improve productivity
- use AI in a practical, human-centered way

### 2. Custom software services

Reva provides end-to-end software services for clients who need custom-built digital products, such as:

- web applications
- mobile applications
- internal tools
- AI-powered applications
- workflow automation systems
- dashboards, portals, and business platforms

## Service Positioning

The services side of Reva Infotech should be described with language similar in spirit to leading custom software firms, but rewritten for Reva’s identity.

### Primary service message

We design and build custom software applications that help businesses move faster, serve customers better, and operate more efficiently.

### Supporting service themes

- modern software for modern business problems
- custom applications built around real workflows
- product thinking, design, engineering, and delivery under one roof
- scalable systems built for growth, speed, and usability

Reva Infotech should position itself as a fit for organizations facing problems like:

- we need a custom platform, not another generic SaaS tool

- we want to add AI to our product in a practical way

## Template Intent For index2.html

The current `index2.html` should be understood as a **high-end visual template** that can support both:

- a product-led Reva homepage
- a service-led Reva Infotech company homepage

That means the template can be adapted into:

### Option A. Reva product site

Focus:

- personal AI application
- productivity and memory tools
- individual user value
- download and signup CTA

### Option B. Reva Infotech company site

Focus:

- custom software development company
- service offerings
- case studies and proof
- consultation and project call CTA

## Recommended Adaptation For Reva Infotech

If `index2.html` is being used for the company website rather than only the product site, the sections should evolve into this structure:

1. Hero
2. Client trust / social proof
3. Services overview
4. How Reva works
5. Product and service capabilities
6. Why teams choose Reva
7. Privacy / quality / engineering trust block
8. Client testimonials
9. Final CTA
10. FAQ
11. Footer
